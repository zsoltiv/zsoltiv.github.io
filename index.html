Git repository készítése és feltöltés Githubra

Helyi repository készítése

git init

Létrehoz egy helyi repository-t a jelenlegi mappában.

Commitolás

git add

Ezzel a paranccsal fájlokat "stagelhetünk", avagy adhatunk hozzá a commitolandó fájlok listájához. Az -a opcióval automatikusan stagel minden módosított fájlt.

git commit

Ez a parancs gyakorlatilag elmenti a fájlokat a jelenlegi tartalmukkal, hasonló egy mentés gombhoz.

Pusholás

Ahhoz, hogy a munkánkat feltölthessük Githubra, létre kell hoznunk egy repository-t az oldalon.

git remote add [név] [URI]

Hozzáad egy repository-t a remote repository-k listájához.

git push [név] [ág]

Feltölti a legutolsó commitot a név remote ág ágára (általában master vagy main).
